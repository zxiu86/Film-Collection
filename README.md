## 🛡️ نظام الأمان - درجات الخطورة

| المستوى | اللون | الوصف | الملفات |

| **متوسط** | 🟡 | يمكن التعديل بوعي | `movie.js`, `search.js`, `stats.js` |
| **آمن** | 🟢 | تعديل بحرية | `slider.js`, `infinite-scroll.js`, `account.js`, `messages.js`, `notifications.js`, `suggestions.js` |

## ⚙️ كيفية التشغيل

### 1. المتطلبات الأساسية
- أي متصفح حديث (Chrome, Firefox, Edge, Safari)
- دعم JavaScript (ES6+)

### 2. التشغيل المحلي
```bash
# باستخدام Live Server (VS Code)
1. افتح المجلد في VS Code
2.右键点击 cine.html
3. اختر "Open with Live Server"

# أو باستخدام Python
python -m http.server 8080
# ثم افتح http://localhost:8080/cine.html
```

3. الدخول كأدمن

· اسم المستخدم: zizo
· سيظهر زر الأدمن تلقائياً
